CLASSIFICATION DES DÉTENUS
1933–1945

Système de marquage SS • Camps de concentration

Présentation

Dans les camps de concentration nazis, chaque détenu portait un insigne textile cousu sur sa tenue. Ce système de marquage permettait aux SS d’identifier immédiatement la catégorie du prisonnier, son origine, et parfois son statut particulier. Les triangles de couleur, les lettres et les combinaisons formaient un code visuel strict, appliqué dans l’ensemble du système concentrationnaire.

Les triangles de couleur

  • Rouge : prisonniers politiques (résistants, communistes, opposants)
  • Vert : criminels de droit commun
  • Bleu : apatrides, émigrés
  • Violet : Témoins de Jéhovah
  • Rose : hommes homosexuels
  • Noir : « asociaux » (Roms, vagabonds, prostituées, handicapés)
  • Marron : Roms (selon les camps)
  • Jaune : Juifs (souvent combiné avec un autre triangle)

Combinaisons pour les détenus juifs

Les Juifs portaient généralement deux triangles superposés formant une étoile :

  • Jaune + rouge : Juif prisonnier politique
  • Jaune + vert : Juif criminel
  • Jaune + bleu : Juif apatride

Marquages supplémentaires

  • Point noir : récidivistes
  • Bande noire : détenus de la compagnie disciplinaire
  • Lettres : nationalité (F = Français, P = Polonais, S = Espagnol…)
  • Symboles spéciaux : militaires, prisonniers « Ia », cas particuliers

Fonction du système

Ce système de classification permettait aux SS d’organiser le travail forcé, de hiérarchiser les détenus, d’identifier les groupes à surveiller de près, et de stigmatiser les catégories jugées « indésirables ». Il est devenu l’un des symboles les plus marquants de la déshumanisation pratiquée dans les camps.
